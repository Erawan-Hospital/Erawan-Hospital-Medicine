# ระบบแจ้งเตือนยาใกล้หมดอายุและคลังยา — โรงพยาบาลเอราวัณ จังหวัดเลย

เว็บนี้อ่านข้อมูลจริงจาก **Google Sheet** ผ่าน **Google Apps Script** และ **Vercel**
เมื่อแก้ไขข้อมูลในชีต เว็บจะอัปเดตอัตโนมัติ (ดึงข้อมูลใหม่ทุก 5 นาที และมีปุ่ม “รีเฟรชข้อมูล”)

```
Google Sheet  ──►  Apps Script (Web App, ส่วนตัว)  ──►  Vercel /api/data  ──►  เว็บ (index.html)
   ข้อมูลจริง          คืนค่าเป็น JSON                     ซ่อน URL + แคช           ดึงทุก 5 นาที + ปุ่มรีเฟรช
```

## ขั้นตอนที่ 1 — เตรียม Google Sheet

สร้าง **4 แท็บ (Sheet tabs)** ในไฟล์ Google Sheet ของคุณ แถวแรกของทุกแท็บคือ **หัวคอลัมน์**
(ชื่อหัวคอลัมน์เป็นภาษาไทยหรืออังกฤษก็ได้ — สคริปต์จับคู่ให้อัตโนมัติ ดูรายการชื่อที่รองรับใน `apps-script/Code.gs`)

### แท็บ `Medicines` (ยาคงเหลือในคลัง)
| รหัสยา | ชื่อยา | หน่วย | คลัง | เลขล็อต | วันหมดอายุ | คงเหลือ |
|--------|--------|-------|------|---------|------------|---------|
| YA-01001 | Paracetamol 500 mg | เม็ด | คลังใหญ่ | LOT-A100 | 2026-09-30 | 320 |

### แท็บ `Receiving` (ประวัติการรับเข้า)
| วันที่ | เวลา | ชื่อยา | หน่วย | จำนวน | เลขล็อต | ผู้รับเข้า | แหล่งที่มา |
|--------|------|--------|-------|-------|---------|-----------|------------|
| 2026-07-10 | 08:30 | Paracetamol 500 mg | เม็ด | 200 | LOT-A100 | ภญ.สมศรี ใจดี | องค์การเภสัชกรรม (GPO) |

### แท็บ `Dispensing` (ประวัติการส่งออก)
| วันที่ | เวลา | ชื่อยา | หน่วย | จำนวน | เลขล็อต | ผู้ส่งออก | ปลายทาง |
|--------|------|--------|-------|-------|---------|-----------|---------|
| 2026-07-11 | 13:20 | Amoxicillin 500 mg | แคปซูล | 60 | LOT-A200 | นางสาวอรทัย พรมมา | ห้องจ่ายยา OPD |

### แท็บ `Users` (บัญชีเข้าสู่ระบบ) — *ไม่บังคับ*
| ชื่อ | ชื่อผู้ใช้งาน | รหัสผ่าน | สิทธิ์ |
|------|--------------|----------|--------|
| ผู้ดูแลระบบ | admin | (ตั้งรหัสเอง) | admin |
| เจ้าหน้าที่เภสัชกรรม | staff | (ตั้งรหัสเอง) | staff |

> **สำคัญเรื่องวันที่:** ให้ช่องวันหมดอายุ/วันที่ เป็น **เซลล์ชนิดวันที่จริง** ของ Google Sheet
> (ไม่ใช่ข้อความ พ.ศ.) สคริปต์จะแปลงเป็นรูปแบบ `YYYY-MM-DD` ให้ และเว็บจะแสดงเป็น พ.ศ. เอง
>
> **สิทธิ์ (role)** ใส่ได้แค่ `admin` หรือ `staff` — admin เห็นเมนูผู้ดูแล/ตั้งค่าเพิ่ม
>
> ⚠️ รหัสผ่านในแท็บ Users เก็บเป็นข้อความธรรมดา ควรจำกัดสิทธิ์การเข้าถึงไฟล์ชีตให้เฉพาะผู้ดูแล

ถ้าโครงสร้างชีตเดิมของคุณต่างจากนี้ — ส่งหัวคอลัมน์มาให้ผมปรับการจับคู่ใน `Code.gs` ได้เลย

## ขั้นตอนที่ 2 — ติดตั้ง Google Apps Script

1. เปิด Google Sheet → เมนู **ส่วนขยาย (Extensions) → Apps Script**
2. ลบโค้ดเดิมทั้งหมด แล้ววางเนื้อหาจากไฟล์ [`apps-script/Code.gs`](apps-script/Code.gs) → กด **บันทึก**
3. กด **Deploy → New deployment** → เลือกชนิด **Web app**
   - **Execute as:** Me (ตัวคุณเอง)
   - **Who has access:** Anyone
4. กด **Deploy** แล้ว **คัดลอก URL** ที่ลงท้ายด้วย `/exec` — นี่คือค่า `SHEET_ENDPOINT`

> ชีตยังคง **เป็นส่วนตัว** — มีเพียงสคริปต์นี้ (ที่รันในนามคุณ) ที่อ่านข้อมูลได้
> ทดสอบได้โดยเปิด URL `/exec` ในเบราว์เซอร์ ควรเห็น JSON ของข้อมูล

## ขั้นตอนที่ 3 — Deploy ขึ้น Vercel

โฟลเดอร์นี้ถูกเตรียมเป็น Git repo ไว้แล้ว วิธีที่ง่ายที่สุด (ไม่ต้องติดตั้งอะไรในเครื่อง):

**A. อัปโหลดขึ้น GitHub**
1. สร้าง repository ใหม่ (private) บน github.com
2. ในโฟลเดอร์นี้ รันคำสั่ง (แทน `<URL>` ด้วยลิงก์ repo ของคุณ):
   ```bash
   git remote add origin <URL>
   git branch -M main
   git push -u origin main
   ```

**B. Import เข้า Vercel**
1. เข้า [vercel.com](https://vercel.com) → **Add New… → Project** → เลือก repo ที่เพิ่ง push
2. Framework Preset: **Other** (เว็บ static + ฟังก์ชัน `/api` อัตโนมัติ) → กด **Deploy**

**C. ตั้งค่า Environment Variable (สำคัญ)**
1. ในโปรเจกต์ Vercel → **Settings → Environment Variables**
2. เพิ่มตัวแปร:
   - **Name:** `SHEET_ENDPOINT`
   - **Value:** URL `/exec` จากขั้นตอนที่ 2
   - Environment: Production (และ Preview ถ้าต้องการ)
3. ไปที่แท็บ **Deployments → … → Redeploy** เพื่อให้ค่าใหม่มีผล

เสร็จแล้ว! เปิดลิงก์ที่ Vercel ให้ (`https://<ชื่อโปรเจกต์>.vercel.app`) แล้วเข้าสู่ระบบ

## การเข้าสู่ระบบ

- ถ้ามีแท็บ **Users** ในชีต: ใช้ชื่อผู้ใช้/รหัสผ่านจากชีต
- ถ้าไม่มี: ใช้บัญชีสำรองในตัว **admin / admin1234** และ **staff / staff1234**
  (แนะนำให้เพิ่มแท็บ Users และตั้งรหัสผ่านของคุณเอง)

## การอัปเดตข้อมูล

- เว็บดึงข้อมูลใหม่จากชีต **อัตโนมัติทุก 5 นาที** และเมื่อกดปุ่ม **🔄 รีเฟรชข้อมูล**
- แถบสถานะบน Dashboard จะเป็น 🟢 เขียว = เชื่อมต่อสำเร็จ, 🔴 แดง = เชื่อมต่อไม่ได้ (จะแสดงข้อมูลตัวอย่างชั่วคราว)
- Vercel แคชผลลัพธ์ ~60 วินาที เพื่อลดภาระ Apps Script
